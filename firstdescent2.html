First Descent

First Descent is a dedicated rental and shop for skiers and snowboarders with a great selection of brands.

My role

In this redesign of the current website, I have created a new visual identity and a prototype using Figma. Icons are med in Illustrator.

Approach

The design is based on an iterative process including research, creating personas, user flows, wireframes and a hi-fi prototype.

The design was developed according to conventional user interface design guidelines. It should be accessible to all users, responsive on different devices and easy to navigate. The interface has a cohesive graphic profile with an adventurous vibe and a freeride skiing feel.
