Lument

The Lumentin® concept renders perfect negative contrast in CT-examinations of the small intestine formerly only possible with MR imaging. Examinations can be made quicker, cheaper, and with a minimum of discomfort for the patients. Lumentin® is also a strong candidate as a universal contrast agent for gastrointestinal imaging. Sales in the European Union are planned to start in 2023.

My role

In collaborations with Lument, I have re-designed and developed a new website and created a new visual identity. I have been in charge of the end-to-end design process doing research, creating wireframes, design guidelines and a hi-fi prototype. The website is developed using TypeScript and a headless CMS system.

Approach

The design is based on an iterative process including meetings and workshops with Lument to create a shared image of how to design and develop the website.

In order for Lument to maintain and update the website by themselves the website is developed using wordpress, a headless CMS, so that they can manage the content without needing any coding skills.

The website is developed according to conventional user interface design guidelines. It's responsive on different devices.
